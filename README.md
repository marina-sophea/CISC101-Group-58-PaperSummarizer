This repository contains the final Paper Summarizer system created for Set 2.

- **system_prompt.md**  
  The complete system prompt generated using the meta-prompt from Task 1.

  - 05_citation_extractor.md

This repo mirrors the Travel Planner project structure and stores the system-level prompt design for the Paper Summarizer application.
